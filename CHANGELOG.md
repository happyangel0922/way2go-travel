# 更新日誌

所有對 Way2Go Travel 的重要更改都將記錄在此文件中。

格式基於 [Keep a Changelog](https://keepachangelog.com/zh-TW/)，
版本遵循 [Semantic Versioning](https://semver.org/lang/zh-TW/)。

## [1.0.0] - 2026-01-04

### 新增

- ✨ 初始版本發布
- 🎨 現代電光藍配色方案
- 🧭 固定導航欄，包含品牌 Logo 和 CTA 按鈕
- 🖼️ 主視覺區，配有高質量城市夜景背景
- 🎯 功能閃卡區塊，展示四種旅遊方式
- 🌍 城市推薦區塊，展示四個熱門目的地（東京、首爾、曼谷、新加坡）
- ⭐ 用戶評價區塊，展示真實用戶的旅遊體驗
- 📱 完全響應式設計，支持所有設備尺寸
- ✨ 平滑的動畫效果（淡入、懸停浮動、滾動揭示）
- 🎭 深色模式支持
- 📊 SEO 優化
- 🚀 高性能構建，使用 Vite 和 Tailwind CSS 4

### 技術

- React 19 + TypeScript
- Tailwind CSS 4
- Vite 構建工具
- Framer Motion 動畫
- shadcn/ui 組件庫
- Wouter 路由管理

### 文檔

- 完整的 README.md
- 詳細的部署指南（DEPLOYMENT.md）
- 貢獻指南（CONTRIBUTING.md）
- MIT 授權
